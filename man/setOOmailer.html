NAME

setOOmailer − Configure LibreOffice and Open Office to use the GNOME preferred email application

SYNOPSIS

setOOmailer

DESCRIPTION

This script will attempt to set the current user’s LibreOffice and Open Office configurations to use the GNOME preferred email application for "Send File" functions. It is intended to run at login.

OPTIONS

None

SEE ALSO

gnome-gmail(1)

BUGS

The script does not report errors.
